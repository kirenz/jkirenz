The xaringan package is an R Markdown extension based on the JavaScript library remark.js to generate HTML5 presentations in different themes (Xie 2019).

You can learn more about the usage of the xaringan package from this excellent documentation, which is actually a set of slides generated from xaringan.

Xie, Y. (2019). Xaringan: Presentation Ninja. CRAN
